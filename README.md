# PixelRush ⚡

A browser-based infinite side-scrolling platformer game 
built from scratch using HTML5 Canvas and vanilla JavaScript.

## Features
- 🎮 Smooth running, jumping and physics
- 🌍 Infinite world generation
- 👾 Ground enemies with rhythmic wave spawning
- 🐦 Flying bird enemies
- 🪙 Coin collection system
- ❤️ 3 lives system
- 🏆 Leaderboard with top 10 scores
- 📷 Scrolling camera follows player
- ✨ Animated star background

## Controls
- Arrow Keys / WASD — Move
- Space / W / Up — Jump

## Built With
- HTML5 Canvas
- Vanilla JavaScript
